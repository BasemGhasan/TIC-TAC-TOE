# Tic-Tac-Toe Game

A simple Tic-Tac-Toe game built with React and TypeScript.

## Features

- Play against another player
- Win and draw detection
- Responsive design
- Reset button with animation

## Installation

1. Clone the repo
2. Run `npm install`
3. Run `npm run dev`
4. Open http://localhost:3000

## Technologies

- React
- Next.js
- TypeScript
- CSS
